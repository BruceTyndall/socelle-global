## A3 — Copy Refinement Rules

| RULE | STANDARD |
| --- | --- |
| One claim per section | Each section makes one strong claim and defends it. Not three medium claims and no defense. The comparison table makes one claim — "Socelle is the only platform built for this" — and proves it with three competitors. That structure is the standard. |
| Shorter is always the premium signal | Hero: 8 words maximum. Subheadline: 20 words maximum. Body section: 60 words maximum. FAQ answer: 3 sentences maximum. If it is longer, it is not done. |
| Specificity over abstraction | Replace every abstract benefit with a specific fact. "Better distribution" → "Your storefront live in 3 to 5 days." "Hundreds of brands" → "Every verified brand on one invoice." The more specific, the more trustworthy. |
| No SaaS or startup language | Forbidden: streamline, leverage, unlock, empower, scalable, ecosystem, seamless, frictionless, game-changing, solutions, best-in-class, next-level. Acceptable: platform, marketplace, infrastructure, relationship, account, verified, professional, direct. |
| Authority without hedging | Remove all "we believe," "we think," "we hope," "our goal is to." State claims. "We believe in transparency" becomes "Our pricing is disclosed at application approval." The claim is stronger without the hedge. |
| SEO keyword presence is natural, not forced | Primary keywords appear in H1 and the first 100 words of body copy. They appear because the sentence required them, not because they were inserted. If a keyword makes the sentence awkward, rephrase the sentence, not the keyword. |
| Write for global English | No idioms, no US-only references, no colloquialisms. Every sentence should be readable in London, Toronto, Sydney, Singapore, and Dubai without cultural translation. Test: if it would appear in The Economist, it passes. |

## B3 — Copy Quality Standard
Every page agent and Agent 6 uses this as the pass/fail test before submission at any gate.

| ELEMENT | STANDARD |
| --- | --- |
| Hero (H1) | 8 words or fewer. One claim. No sub-clauses. States a competitive or strategic truth. Reads in 1.5 seconds. |
| Subheadline | 20 words or fewer. Names the audience or the benefit. Does not repeat the headline. Does not over-explain. |
| Section intro (H2) | 10 words or fewer. Not a sentence — a declaration. Reads like a chapter title, not a slide header. |
| Body paragraph | 50 words or fewer. Opens with a claim. Provides one piece of evidence. Closes with the implication. |
| Feature description | 2 sentences maximum. Sentence 1: the outcome the user experiences. Sentence 2: how the feature delivers it. |
| FAQ answer | 3 sentences maximum. Direct. No hedging. No footnotes. If it requires more than 3 sentences, the question is too broad. |
| CTA label | 2 to 4 words. Action verb. Specific to the destination. No generic "Learn more" or "Get started" without context. |
| Trust bar / micro-copy | 3 items maximum. Each item 3 words or fewer. Factual, not aspirational. |

## B4 — Premium Clarity Standard
Premium copy is not just short copy. It is confident, specific, and generous to the reader's intelligence. The following test distinguishes premium from competent.

| Premium test | Read the copy and ask: does this sound like it belongs on the website of a brand that has been doing this for twenty years? Or does it sound like a startup explaining its idea? Premium brands do not explain — they state. They do not justify — they demonstrate. |
| --- | --- |
| Specificity test | Replace every abstract word with the most specific word available. "Better distribution" fails. "Your storefront live in 3 days" passes. "Powerful analytics" fails. "Purchasing benchmarks by category" passes. |
| Confidence test | Remove every hedge. "We believe," "we aim to," "our goal is to," "we try to" all fail. State the claim directly. If the claim is not defensible without a hedge, the claim is wrong and should be rewritten, not softened. |
| Economy test | Count the words in the hero subheadline. If it is over 20 words, cut. Count the words in the body paragraph. If it is over 50, cut. Premium is restraint. Restraint is trust. |
| Translation test | Read the copy as if you are translating it to German, Japanese, or Portuguese. Does the meaning survive? If the sentence contains idioms, cultural references, or syntactic inversions that would break in translation, rewrite it. |

## C8 — Rules for Balancing SEO Strength with Premium Tone

| RULE | APPLICATION |
| --- | --- |
| Keywords serve sentences, not the reverse | A sentence written for its keyword is unreadable. Write the best sentence for the user. Then verify the keyword is present. If it is not, restructure the sentence to include it naturally. Never inject a keyword into a sentence that did not require it. |
| Premium brands rank on entity coverage, not repetition | Repeating "professional beauty wholesale" ten times on a page is not SEO — it is a penalty risk. Covering the entity cluster (brands, buyers, verified, licensed, tiered pricing, single checkout) across the page naturally earns the ranking. |
| Length is not a ranking signal on its own | Long pages rank when they are comprehensive. Short pages rank when they are authoritative. The Pricing page should be short and authoritative. The Wholesale Beauty Guide should be long and comprehensive. Do not add words to a page to "improve SEO." |
| User signals are the real ranking factor | Click-through rate, scroll depth, and bounce rate determine ranking position more than keyword density. Copy that makes users stay, scroll, and convert sends stronger ranking signals than copy stuffed with keywords. |
| The homepage is a brand page, not a category page | The homepage should target branded and semi-branded search. The vertical landing pages (/for-salons, /for-medspas) capture high-volume unbranded commercial terms. Do not force every keyword onto the homepage. |

## C9 — Keyword Stuffing and Thin Content Prevention

| RISK | PREVENTION RULE |
| --- | --- |
| No paragraph that exists for SEO only | Every paragraph must serve the reader first. If removing a paragraph does not reduce the user's comprehension of the page, it should not exist. Thin content is a ranking risk, not an advantage. |
| No exact-match keyword in more than 2 consecutive sentences | If a keyword or phrase appears in the same form in two adjacent sentences, rephrase one of them. Natural writing uses synonyms and related terms. |

# D — Multilingual Copy Standard
Professional beauty is a global industry. SOCELLE’s premium positioning must survive translation without losing tone, specificity, or strategic clarity. This section defines the writing standards that enable clean multilingual adaptation — not by limiting the English copy, but by writing it in a way that translates well from the start.
## D1 — Multilingual Copy Rules

| RULE | DETAIL |
| --- | --- |
| Write for the sentence, not the voice | Voice is cultural. Premium tone is structural. Short sentences, specific nouns, active verbs, and no idioms produce premium copy that translates across languages without losing its quality signal. |
| One idea per sentence | Complex sentences with multiple subordinate clauses are harder to translate accurately. Each sentence conveys one complete idea. Translators can recompose naturally in the target language from simple source sentences. |
| Avoid progressive constructions | "We are building" translates awkwardly in many languages. "We build" is cleaner and stronger. Prefer simple present and simple past over progressive forms where the meaning is equivalent. |
| Avoid passive voice in CTAs | CTA text in passive voice ("Applications are being reviewed") signals weakness in English and translates inconsistently. Active voice CTAs ("Apply now," "Join free") translate cleanly and carry authority. |
| Never use idioms in hero copy | Idioms are culturally untranslatable. "Where professional beauty moves" works in English because "moves" carries both physical and market momentum connotations. This meaning is lost in French, German, and Mandarin. Reserve idioms for brand identity documents, not translatable web copy. |
| Spell out numbers at sentence openings | "3 steps" in English translates cleanly, but starting a sentence with a numeral is grammatically inconsistent across languages. "Three steps" is universally cleaner in translation. |

## D2 — Sentence-Length and Syntax Rules

| Hero H1: maximum 8 words | Short headlines translate with the least distortion. 8 words in English becomes 6 to 12 words in German and 10 to 14 characters in Mandarin. Headlines designed at 8 words give translation teams room to work. |
| --- | --- |
| Hero subheadline: maximum 20 words | 20-word subheadlines translate into manageable equivalents in all major European and Asian languages. Beyond 20 words, meaning fragmentation is high. |
| Body paragraph: maximum 50 words | 50-word paragraphs translate into 40 to 70 words in most target languages. Paragraphs over 50 words often require structural splits in translation, which changes the reader's experience. |
| FAQ answers: maximum 60 words | FAQ answers in English often run long. 60 words is the ceiling. Translators working from tighter source material produce tighter translations. |
| Avoid: relative clauses stacked on relative clauses | Sentences like "the platform that brands who sell wholesale to operators who are licensed use to manage their accounts" are untranslatable at quality. Restructure: "Brands use SOCELLE to manage wholesale accounts with licensed operators." |

## D4 — Phrases to Avoid
The following phrases translate poorly, carry cultural specificity, or reduce premium signal in non-English markets. They are eliminated from all copy going forward.

| RULE | DETAIL |
| --- | --- |
| "Treatment room" | Understood in US and UK professional beauty. Not universally understood in French, German, or Asian markets. In global copy, use "professional service environment" or "licensed service business." |
| "Everyone wins" | Culturally US-specific framing. In many markets this reads as hyperbole or as a child's sports-day motto. Replace with a structural claim about the two-sided value. |
| "Gray market" | A technical trade term that translates inconsistently. In some markets it implies different legal definitions. Use "non-verified resellers" or "retail gray-market channels" with a brief definition on first use. |
| "On our roadmap" | Startup jargon. International readers may not understand the software-product connotation. Avoid in public-facing copy. If a feature is coming, give a timeframe or remove the reference. |
| "Best-in-class" | A phrase that has lost all meaning in any language. Remove from all copy. |
| "Seamless" | Used in 40% of all SaaS copy. Meaningless and untranslatable in the intended sense. Replace with the specific friction point it removes. |
| "Game-changing" | Hyperbolic. Untranslatable as intended. Remove entirely. |
| "Streamline your workflow" | Operational jargon that does not survive into French, Japanese, or Portuguese with its intended meaning. Name the specific operation being simplified. |

## D5 — Premium Global-English Standard
Global English is the writing standard used by The Economist, Bloomberg, and premium global brands operating in multiple markets. It is characterized by: clear syntax, active voice, specific nouns, no idioms, and confident declarative sentences.

| The standard source | The Economist style guide is the reference standard. Not because SOCELLE is a publication, but because it represents the highest level of global-English clarity. If a sentence could appear in The Economist without revision, it passes the global English test. |
| --- | --- |
| Active, not passive | Active voice: "SOCELLE verifies every brand." Passive voice: "Every brand is verified by SOCELLE." The active form translates better, scans faster, and signals confidence. |
| Nouns over adjectives | Premium brands favor specific nouns over descriptive adjectives. "A dedicated storefront" is weaker than "Your storefront." "Powerful analytics" is weaker than "purchasing benchmarks by category." |
| Declarative sentences over questions | Marketing copy that opens with questions ("Tired of managing six portals?") is common in mid-market brands. Premium global brands open with declarations. Questions can appear in FAQs. They do not open sections. |
| Precision in numbers | "Hundreds of brands" is vague and regional in its connotation. "More than 200 verified brands" is precise, globally readable, and credibility-building. Use specific numbers wherever they are known. |

## D6 — Multilingual SEO Considerations

| RULE | DETAIL |
| --- | --- |
| hreflang implementation | When additional language versions launch, implement hreflang tags on all pages. This prevents duplicate content penalties across language versions and directs search engines to the correct regional variant. |
| Keyword research per market | Keywords do not translate literally. "Professional beauty wholesale" in English does not have a direct French or German equivalent with the same search volume. Each language version requires its own keyword research before launch. |
| URL structure for multilingual | Use subdirectory structure: socelle.com/fr/, socelle.com/de/. Do not use country-code TLDs for the first expansion phase. Subdirectories inherit the root domain’s authority. |
| Metadata must be translated, not transcribed | Meta titles and descriptions must be written in the target language using native SEO keyword research, not translated from English. A translated meta title rarely matches the search language of the target market. |
| English copy is the canonical source | All translation work begins from the approved English copy. Changes to English copy trigger re-translation. This is enforced through the controlled terminology list in Section D3. |

## D7 — Global Consistent vs Market-Localized Content

| Globally consistent (do not localize) | Platform mechanics, pricing structure, commission model, tier names (Active, Elite, Master), brand application process, verification model, comparison table, About page argument, Intelligence Hub description. These are product truths that do not change by market. |
| --- | --- |
| Localize by market | Regulatory language (licensing requirements vary by jurisdiction), currency (USD for US, local currency for international), specific vertical categories that are prominent in some markets and absent in others (e.g., ayurvedic or K-beauty specific language), social proof with market-specific brand or operator names. |
| Persona copy localization | The core persona (licensed service-business owner) is global. The specific language around professional credentials and business categories varies by market. "Licensed cosmetologist" is US-specific; "registered beauty therapist" is UK-specific. Localize the credentialing language, not the persona logic. |
| Photography and imagery | US-launched site can use US-market imagery. International expansion requires market-appropriate visual representation. Flag this for the Phase 2 imagery brief — it is not a copy issue but it travels with copy decisions. |

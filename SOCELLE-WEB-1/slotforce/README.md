# SLOTFORCE Monorepo

Monorepo foundation for SLOTFORCE backend and shared logic.

## Structure

- `apps/functions`: Firebase Cloud Functions (TypeScript)
- `packages/shared`: Shared schemas/types
- `packages/gap-engine`: Pure gap detection logic

## Commands

- `npm run build`
- `npm run test`
- `npm run lint`
- `npm run typecheck`

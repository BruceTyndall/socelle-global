Design System v1.0
SOCELLE ZERO
The Neutral Stage.
A unified design system for Web + Flutter mobile. Brands bring the color. SOCELLE is the stage. Every surface is neutral enough that any brand identity placed on it immediately commands attention.
Web — React + Tailwind CSS
Mobile — Flutter + Material 3
Current problem: 6 competing color systems in the codebase (pro-*, brand-*, intel-*, edu-*, cocoa, mn-*). Warm champagne, terracotta, and gold accents compete directly with beauty brand identities. Brands with warm palettes (Tatcha, Aveda, La Mer) will clash — not pop.
Solution: One neutral system. Near-white background, near-black text, single desaturated chrome accent. Color appears only in: (1) brand content/imagery, (2) data signals. The platform disappears — the brands appear.
Core Neutrals — The Stage
Parchment — Page Background
#F8F8F6 — rgba(248,248,246)
Near-white with 0.7% warm tint. Works against every brand palette.
Parchment Alt — Section BG
#F2F2F0
Alternate section background — barely perceptible difference.
Glass Panel — Floating Containers
rgba(255,255,255,0.70) + blur(24px)
Primary surface for all floating panels, cards, and nav.
Void — Dark Panel (1 per page max)
#111112 — near-pure black, 0.3% cool
Used ONCE per page for primary metrics / social proof.
Ink — Primary Text
#0D0D0C — near-pure black, 0.5% warm
Reads as black but has a hair of warmth to pair with parchment.
Chrome — UI Accent Only
#4A5568 — cool blue-gray, heavily desaturated
Used ONLY for interactive UI chrome (links, active states, focus). Never decorative.
Data Signals — Data Layer Only
Signal Up — Growth
#15803D + rgba(21,128,61,0.10) bg
Data layer only. Never used as a design color. Appears on intel cards, stat badges.
Signal Warn — Watch
#B45309 + rgba(180,83,9,0.10) bg
Flat/declining attention. Data context only.
Signal Down — Decline
#B91C1C + rgba(185,28,28,0.10) bg
Negative movement. Data context only.

Implementation Roadmap
6 systems → 1 system.
The existing codebase has 6 competing color identities. This is the migration path. Portals retain backward-compatible aliases during transition.

3 Backgrounds — Atmospheric System
/* index.css — replace flat bgs */ /* Home / public pages */ --bg-home: radial-gradient( ellipse 60% 50% at 35% 20%, rgba(240,240,238,1.0) 0%, transparent 65%), #F8F8F6; /* Intelligence Hub */ --bg-intel: radial-gradient( ellipse 55% 50% at 70% 30%, rgba(238,240,240,0.8) 0%, transparent 60%), #F8F8F6; /* Education / dark content */ --bg-edu: var(--parchment); /* Rule: gradients are near-imperceptible. Max delta from parchment: 6% lightness. Brands ARE the color. */
4 Flutter — What to Build First
Flutter
Start with socelle_theme.dart — the ThemeData + custom extensions (see companion file). This is the single source of truth for mobile.
Flutter
Build GlassPanel, PillButton, SignalChip, SegmentedPillControl as shared widgets in lib/ui/.
Flutter
Bottom nav: NavigationBar (M3) with backgroundColor: transparent, glass scaffold, white indicator.
Flutter
Font: embed General Sans in assets/fonts/. Declare in pubspec.yaml. JetBrains Mono for data values.
Flutter
Use useMaterial3: true with ColorScheme.fromSeed(seedColor: Color(0xFF4A5568)) — neutral seed generates neutral M3 scheme.

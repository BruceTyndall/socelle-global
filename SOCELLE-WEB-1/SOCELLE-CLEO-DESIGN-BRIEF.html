Rule 01
Atmospheric Gradient Background
The page background is never flat. It uses radial gradients — warm champagne at the focal point, cooler sand/stone at edges — creating depth and air before anything is placed on top.
Replace flat #F6F4F1 with radial gradient. Each page gets its own gradient personality.
Rule 02
Glassmorphism Architecture
Every container — nav, panels, cards, tags — floats as frosted glass over the background. background: rgba(255,255,255,0.55) + backdrop-filter: blur(24px) is the signature.

Rule 03
Floating Navigation Pill
The nav is never a full-width bar. It floats as a centered pill — border-radius: 40px — with the logo pill on the left and a CTA pill on the right, all separate floating objects.

Rule 04
Aggressive Pill Geometry
Border-radius is pushed to its logical extreme. Panels use 40px. Cards use 28px. Tags and controls use 20px. Buttons are circles. Nothing has a sharp corner.

Rule 05
Frosted Category Tags

Rule 06
One Dark Panel Rule
Cleo uses exactly one dark panel per page — a deep glass surface with rgba(20,25,30,0.88) that creates contrast and visual hierarchy. Never more than one per page.

Rule 07
Segmented Pill Controls
Filter bars and tab switchers use a pill container with an inner active pill that floats between states. Never underline tabs, never colored borders — only the floating pill moves.
Replace all border-b tabs with pill segmented controls. Apply to every filter/tab in all portals.
Rule 08
Circle Arrow CTAs
Cards don't have rectangular "View" buttons. Instead, a frosted circle with an arrow lives in the bottom-right corner. It's small, unobtrusive, and reveals intent on hover.

Implementation Plan
What to
change.
Ordered by visual impact. These changes apply globally across all 11 public pages — portals follow the same logic with darker, data-denser variants.

1Background System

Replace flat bg-[#F6F4F1] with per-page radial gradient backgrounds
Add 5 gradient presets to Tailwind config: bg-warm-hero, bg-warm-intel, bg-warm-edu, bg-warm-brand, bg-warm-portal
Background bleeds atmospheric color at left/right edges of content panels
Update index.css body to use gradient base — no flat background colors anywhere

2Navigation Redesign

Remove full-width nav bar. Split into 3 floating pill elements: logo, nav links, CTA
Apply backdrop-filter: blur(24px) + bg-white/55 + rounded-[40px] + border border-white/85 to all three
Active nav link gets inner pill treatment — no underlines, no color change
Nav floats over the gradient background — position: sticky, transparent outside the pill

3Panel & Card Architecture

All content sections become floating glass panels: rounded-[40px] + bg-white/55 + backdrop-blur-xl
Content cards: rounded-[28px] — never rounded-lg or rounded-xl
Add subtle box-shadow: shadow-[0_8px_48px_rgba(0,0,0,0.06)] to all panels
Never full-width flush sections — always 48px horizontal padding minimum

4Intelligence Hub Cards

Signal cards become full-bleed image cards with atmospheric gradient backgrounds
8 gradient presets mapped to signal categories (sky=trending, teal=ingredient, rose=treatment, sage=protocol)
Category tag: top-left corner, frosted glass pill, uppercase + tracked, 10px
CTA: frosted circle arrow button bottom-right — replaces text links
Card title centered (large, light weight) + subtitle bottom-left

5Filter & Tab Controls

Replace all border-b tab bars with segmented pill controls globally
Tab container: rounded-[40px] + bg-black/10. Active pill: bg-white/80 + shadow-sm
Filter chips: rounded-[20px], active state uses glass-white with border
Apply to: Intelligence Hub filters, Education category filter, Portal tabs, Brand portal tabs

6Dark Panel + Typography

One dark panel per public page max. Use bg-[rgba(20,25,30,0.88)] + backdrop-blur-xl
Dark panel holds primary social proof / metrics — 4-up stat grid
Body text color: deepen to #3A1A10 (Cleo's warm brown vs current cold #1E252B)
Headings: heavier weight (700), tighter tracking (letter-spacing: -0.03em), larger scale contrast
Eyebrow labels: 11px, 600 weight, 0.12em tracked, uppercase — all section intros
